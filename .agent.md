# Work Notes Project

## Purpose
This is a personal work notes repository for storing and organizing knowledge.

## Content Types
- Concepts and explanations
- Commands and scripts
- Queries and solutions
- Technical references
- Quick tips and tricks

## Agent Instructions
When users ask questions about content in this project:
- Search and find relevant notes
- Provide **short and direct answers**
- Reference the specific file if helpful
- Don't over-explain; be concise
- Help users quickly locate and retrieve information

## Usage
Users will ask questions and expect you to search through the notes and provide quick, relevant answers from the existing content.
