# Work Notes Project

## Purpose
Personal knowledge base for work-related information, organized for quick retrieval and updates.

## Agent Role
You are the primary interface for this project. You will:
- **Answer questions** by searching through notes
- **Update files** when new information is provided
- **Create new files** as needed for organizing content
- **Keep everything simple and direct**
- Provide short, direct answers

## Guidelines
- Keep files simple and organized
- Make it easy to find where to search
- Update files immediately when new information is given
- Create new files for new topics/categories as needed
- Use clear file names that indicate content

## File Organization
- Group related information by topic (dynatrace.md, etc.)
- Use clear headers and sections
- Include code blocks and examples when relevant
- Keep formatting consistent and scannable
